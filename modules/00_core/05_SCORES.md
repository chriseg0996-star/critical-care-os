# 05_SCORES

## General rule

Calculate scores when enough data are available.  
If incomplete, state which variables are missing.

## Common ICU scores

- SOFA
- APACHE II
- SAPS II
- NUTRIC
- CPIS
- ISTH DIC
- PADUA

## Neurocritical scores

- Fisher
- Modified Fisher
- WFNS
- Hunt-Hess
- GRAEB
- NIHSS
- ASPECTS
- ICH score
- FUNC score
- SEDAN
- DRAGON

## Renal

- KDIGO AKI
- CKD G/A staging when data exist

## Output style

Use:
`SOFA: __ puntos.`
`APACHE II: __ puntos.`
`Pronóstico: reservado para la vida y función, condicionado por ___.`

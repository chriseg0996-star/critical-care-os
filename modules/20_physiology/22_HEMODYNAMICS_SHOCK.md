# 22_HEMODYNAMICS_SHOCK

## Scope

Use for shock, perfusion, vasopressors, inotropes, lactate and advanced hemodynamics.

## Shock classification

Consider:
- distributive
- hypovolemic
- cardiogenic
- obstructive
- mixed

## Perfusion markers

Integrate:
- MAP
- lactate
- capillary refill
- urine output
- mental status
- mottling
- venous oxygen saturation when available

## Ultrasound

Interpret:
- LVOT VTI
- cardiac output
- cardiac index
- IVC
- VExUS
- RV/LV function
- pericardial effusion
